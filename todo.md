## 🔧 技术栈确认

### 后端（保持不变）
- **框架**: NestJS 10.x
- **数据库**: PostgreSQL 16 + Prisma ORM
- **缓存**: Redis
- **认证**: JWT

### 前端用户端
- **框架**: Vue 3 + TypeScript
- **状态管理**: Pinia
- **路由**: Vue Router 4
- **构建工具**: Vite
- **UI**: Element Plus / Ant Design Vue

### 管理后台
- **框架**: React 18 + TypeScript
- **UI组件**: Ant Design
- **状态管理**: React Context / Redux Toolkit
- **构建工具**: Vite
